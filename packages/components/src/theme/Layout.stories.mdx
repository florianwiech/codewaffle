# Layout

This Components provides a `ThemeProvider` for `styled-components`.

## Usage

```tsx
import { AppearanceState, Layout } from "./Layout";

export const App = () => (
  <Layout theme={AppearanceState.DARK}>
    {/* ... */}
  </Layout>
)
```
